To run the webapp, ensure that you have all the requirements installed, which are specified in the requirements.txt. It's easiest to run the webapp from the command line. 

1. Open terminal and ensure that you are on a virtual environment that has all the requirements
2. cd "path_to_src_file" 
3. streamlit run app.py

If this doesn't work make sure that you have all the requirements and ensure that you connected to the src file in the project files. All this information is also available in the readme in the project files.
